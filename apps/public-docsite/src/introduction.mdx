# Introduction

FluentPlus is a component library that extends Microsoft Fluent UI v9 with higher-level, opinionated components designed to accelerate development. Rather than building everything from base primitives, FluentPlus provides ready-to-use composite components and common patterns.

**What makes FluentPlus different:**
- **Higher-level abstractions** - Pre-built combinations of Fluent controls
- **Common patterns** - Forms, navigation, dialogs, and layouts out of the box
- **Consistent design** - Seamlessly integrates with Fluent's design system
- **Developer productivity** - Less boilerplate, faster implementation

FluentPlus complements Fluent UI v9 by offering an opinionated layer that handles the common use cases, letting you focus on your application's unique requirements.
